1) Create Project
2) Go to OAuth Consent Page and make project external, with default steps go to next pages
3) Again go back to OAuth page and publish the project to prodcution
4) Go to credentials and create OAuth Client ID > Choose Webapp > Add URI will be localhost:5501> Redirect URIs is this one https://developers.google.com/oauthplayground
5) Go back to Dashboard and enable APIS and services,choose gamil and enable it 
6) Now copy index.html from https://developers.google.com/gmail/api/quickstart/js and paste in your project
7) Update Client ID and API key in the index.html with what u just created in the gmail project and run the index.html
